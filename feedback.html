What is Web Storage?

Modern web browsers let websites remember things — thanks to something called Web Storage.

There are two types:

  • localStorage: Stores data that stays even after you close the browser. Great for saving settings or preferences.
  • sessionStorage: Keeps data only while the page is open. Once you close the tab, it’s gone. Perfect for temporary info.

These tools help websites create smoother and more personalized experiences — without needing to store everything on a server.
